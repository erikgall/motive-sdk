# Motive SDK

> A first-party-quality Laravel SDK for the Motive ELD API

- Expressive, elegant syntax with fluent, chainable methods
- Full support for all 31 Motive API resources
- Strongly-typed DTOs with automatic casting
- Built-in pagination, error handling, and testing utilities
- OAuth and API key authentication support
